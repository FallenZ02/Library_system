## Project Description
MyLibrary+ is a web-based library management system designed for administrators. It allows admins to manage books, members, and track borrowing/return history efficiently with a clean and responsive interface.

---

## Features
- **Admin Dashboard**: Total books, available books, borrowed books, interactive charts, light/dark mode.
- **Manage Books**: Add, edit, delete, search, and filter books.
- **Manage Members**: Add, edit, delete, and search members.
- **Borrowing & Return History**: Track book transactions with borrower info and timestamps.
- **Responsive UI**: Built with Bootstrap 5 and Bootstrap Icons.
